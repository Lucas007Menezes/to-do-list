# 📝 To-Do List (Lista de Tarefas)

Este é um projeto de **Lista de Tarefas (To-Do List)** desenvolvido como parte de um projeto acadêmico utilizando o framework **Django** (Python).

O objetivo do projeto é fornecer uma interface simples e intuitiva para o gerenciamento de tarefas diárias, permitindo que o usuário organize suas atividades de forma eficiente.

---

## 🚀 Funcionalidades (Planejadas/Em Desenvolvimento)

- [x] Estrutura inicial do projeto Django.
- [ ] Criação e edição de tarefas.
- [ ] Exclusão de tarefas.
- [ ] Marcação de tarefas como concluídas.
- [ ] Filtro por status (pendentes/concluídas).
- [ ] Interface responsiva.

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** [Python](https://www.python.org/)
- **Framework Web:** [Django](https://www.djangoproject.com/)

## ⚙️ Como executar o projeto localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

### 1. Clonar o repositório
```bash
git clone https://github.com/Lucas007Menezes/to-do-list.git
cd to-do-list
```

### 2. Criar e ativar o ambiente virtual (Opcional, mas recomendado)
```bash
python -m venv venv
# No Windows:
.\venv\Scripts\activate
# No Linux/macOS:
source venv/bin/activate
```

### 3. Instalar as dependências
```bash
pip install -r requirements.txt
```

### 4. Realizar as migrações do banco de dados
```bash
python manage.py migrate
```

### 5. Iniciar o servidor de desenvolvimento
```bash
python manage.py runserver
```

Acesse o projeto em: `http://127.0.0.1:8000/`
